# mertester
Orchestration command line tool for testing Merlion localnet

## Installation

```bash
$ yarn install
```

## Running

```bash
# development
$ yarn cli -h

# production mode
$ yarn cli:prod -h
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
